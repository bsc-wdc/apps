# PyCOMPSs library and examples

## Dependencies

The dependencies of PyCOMPSs tests and lib can be found in: requirements.txt. To install them issue:

`pip install -r requirements.txt`



## Tests 

To run the pycomss_lib tests issue:


`./run_tests.sh`
